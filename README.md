\# Landmark-Based MRI-to-X-ray Registration



This repository contains the source code developed for the landmark-based registration approach as part of the AI Project at MCI.



\## Repository Structure



\- `src/` – source code

\- `requirements.txt` – Python dependencies



\## Data



The medical imaging data and manually annotated landmarks used in this project cannot be made publicly available due to privacy and data protection restrictions.



Therefore, this repository only contains the source code.



\## Method



The implemented pipeline consists of:



\- Anatomical landmark extraction

\- Landmark verification

\- Landmark consistency analysis

\- Landmark-based registration

\- Registration evaluation



\## Installation



```bash

pip install -r requirements.txt

```
